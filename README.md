```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser

## How to Use

1. **Explore the Graph**: Pan and zoom around the concept graph
2. **Click on Available Blocks**: Green-bordered blocks with play icon are available
3. **Complete Concepts**: Click on a concept to view details and mark it as complete
4. **Unlock New Topics**: Completing prerequisites unlocks new concepts
5. **Track Your Progress**: Watch your XP grow and level up!
